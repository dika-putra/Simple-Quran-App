# Simple-Quran-App
A simple app to show Quran chapter list, chapter info and verses

# Tech Stack:
- Kotlin
- MVVM
- Modular By Layer
- Repository Pattern
- Dagger Hilt
- Paging 3
- Coroutines
- Lifecycle
- Room DB
- Data Binding
- View Binding
- Jetpack Navigation
- Safe Arguments
- Constraint Layout
- Coordinator Layout
- Retrofit 2
- Timber
